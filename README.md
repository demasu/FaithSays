# Faith Says

## Description

This is my entry for the Frame Fatales 2026 Game Jam!

Do you like Simon Says but wish you could follow a cat instead? Well now you can! Introducing Faith Says, the fabulous new game from A Study In Futility Productions!

## Instructions

Faith will show off her moves and you need to follow them! Faith has four actions:

- Walk Left
- Walk Right
- Meow Left
- Meow Right

Remember the order that Faith shows you, and when she's done, repeat those to show her you've been paying attention! See how far you can go before messing up!

## Controls

Up arrow/W    - Walk Left
Down arrow/S  - Walk Right
Left arrow/A  - Meow Left
Right arrow/D - Meow Right
Enter         - Select
Escape        - Go to the main menu

## Running

Download the zip provided either from [the GitHub repo](https://github.com/demasu/FaithSays) or from the [itch.io](https://demasu.itch.io/faith-says) page. Extract the contents to a new folder where you won't lose them. Open that folder and run `Faith Says.exe` and you're good to go!
